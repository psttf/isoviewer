Tool for viewing ISO 2709 files on .Net platform (on MS Windows). Includes API for accessing and writing ISO 2709 files in C++ and C#.

Программа просмотра файлов в формате ISO 2709 для MS Windows (платформа MS .Net), а также библиотеки (API) для программного доступа к файлам в данном формате, написанные на C++ и C#. Поддерживается как чтение так и запись данных.
